Credits & Contact

Wax is a minimal computing (minicomp) project led and maintained by Marii Nyrop and Alex Gil.

It uses open source libraries and frameworks including Jekyll, IIIF, OpenSeaDragon, and ElasticLunr and builds upon work by Peter Binkley, David Newbury, and others.

We welcome and encourage contributibutions. Please get in touch, or fork the project on GitHub. If you encounter any issue with the software, please create an issue or find us in the #minicomp-wax channel within the Code4Lib Slack. (Request an invite here!)

For more on working with Wax, visit our documentation wiki.
